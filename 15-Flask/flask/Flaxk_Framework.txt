⚔️ 3. Difference (Most Important 🔥)
| Feature | WSGI                    | Jinja2              |
| ------- | ----------------------- | ------------------- |
| Type    | Interface/Protocol      | Template Engine     |
| Kaam    | Server ↔ Python connect | HTML dynamic banana |
| Use     | Backend communication   | Frontend rendering  |
| Example | Gunicorn, uWSGI         | Flask templates     |
| Role    | Request handle karta    | UI banata           |
