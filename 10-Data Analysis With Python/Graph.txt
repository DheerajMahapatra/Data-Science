1️⃣ Matplotlib kya hai

Matplotlib Python ki basic plotting library hai.

Use cases:
    Basic graphs
    Custom graphs
    Scientific plotting
    Research



2️⃣ Seaborn kya hai

Seaborn ek high level visualization library hai jo Matplotlib ke upar bani hai.

Advantages:

    Better design
    Statistical graphs
    Less code
    Built-in datasets




3️⃣ Matplotlib vs Seaborn
| Feature       | Matplotlib              | Seaborn                   |
| ------------- | ----------------------- | ------------------------- |
| Level         | Low level               | High level                |
| Design        | Basic                   | Beautiful                 |
| Focus         | General plotting        | Statistical visualization |
| Customization | High                    | Medium                    |
| Code          | Long                    | Short                     |
| Use           | Research, physics, math | Data analysis, ML         |

8️⃣ Simple rule (Interview trick)

Agar interviewer puchhe:

Graph choose kaise karte ho?

Answer:
    Trend → Line plot
    Comparison → Bar chart
    Distribution → Histogram
    Percentage → Pie chart
    Relationship → Scatter plot
    Correlation → Heatmap
    Outliers → Boxplot



9️⃣ Kab Matplotlib aur kab Seaborn

Use Matplotlib when
    Custom graphs banana ho
    Scientific plotting
    Low level control chahiye

Use Seaborn when
    Statistical analysis
    Beautiful graphs
    Data exploration



🔟 Important Interview Question

Q: Agar Seaborn hai to Matplotlib kyun use karte hain?

Answer:
    Seaborn internally Matplotlib use karta hai.
    Matplotlib more flexible hai aur customization jyada deta hai.








📊 Matplotlib Graphs
| Graph          | Function             | Kis type ka data                     | Kab use karte hain               | Example field         |
| -------------- | -------------------- | ------------------------------------ | -------------------------------- | --------------------- |
| Line Plot      | `plt.plot()`         | Numerical vs Numerical (time series) | Trend dekhne ke liye             | Stock market, weather |
| Bar Chart      | `plt.bar()`          | Categorical vs Numerical             | Categories compare karne ke liye | Sales, survey         |
| Horizontal Bar | `plt.barh()`         | Categorical vs Numerical             | Long category names ho           | Business reports      |
| Histogram      | `plt.hist()`         | Single Numerical                     | Distribution dekhne ke liye      | Age, salary           |
| Pie Chart      | `plt.pie()`          | Percentage / Proportion              | Share show karna                 | Market share          |
| Scatter Plot   | `plt.scatter()`      | Numerical vs Numerical               | Relationship dekhna              | Height vs weight      |
| Box Plot       | `plt.boxplot()`      | Numerical                            | Outliers detect karna            | Salary analysis       |
| Area Plot      | `plt.fill_between()` | Numerical vs time                    | Cumulative trends                | Population growth     |
| Stem Plot      | `plt.stem()`         | Discrete numeric                     | Signal processing                | Engineering           |
| Error Bar Plot | `plt.errorbar()`     | Numerical with error                 | Measurement uncertainty          | Scientific research   |



📊 Seaborn Graphs
| Graph        | Function            | Data Type                  | Kab use karte hain     | Field                |
| ------------ | ------------------- | -------------------------- | ---------------------- | -------------------- |
| Scatter Plot | `sns.scatterplot()` | Numerical vs Numerical     | Variable relationship  | ML, Data Science     |
| Line Plot    | `sns.lineplot()`    | Numerical vs Time          | Trend analysis         | Finance              |
| Bar Plot     | `sns.barplot()`     | Categorical vs Numerical   | Mean comparison        | Business             |
| Count Plot   | `sns.countplot()`   | Categorical                | Category frequency     | Survey analysis      |
| Histogram    | `sns.histplot()`    | Numerical                  | Data distribution      | Statistics           |
| KDE Plot     | `sns.kdeplot()`     | Numerical                  | Probability density    | Data Science         |
| Box Plot     | `sns.boxplot()`     | Numerical + Categorical    | Outliers detection     | Salary analysis      |
| Violin Plot  | `sns.violinplot()`  | Numerical + Categorical    | Distribution + density | Research             |
| Strip Plot   | `sns.stripplot()`   | Numerical + Categorical    | Data points show       | ML analysis          |
| Swarm Plot   | `sns.swarmplot()`   | Numerical + Categorical    | Non-overlapping points | Small datasets       |
| Pair Plot    | `sns.pairplot()`    | Multiple numerical columns | Feature relationships  | ML                   |
| Heatmap      | `sns.heatmap()`     | Matrix / correlation       | Feature correlation    | Machine learning     |
| Joint Plot   | `sns.jointplot()`   | Two numerical              | Scatter + distribution | Statistical analysis |
| Facet Grid   | `sns.FacetGrid()`   | Multiple categorical       | Multi plot comparison  | Advanced EDA         |
